# Store Management System

## Description
This project focuses on a sales management system designed to manage a store efficiently.

## Project Goals
- Manage products and sales in a centralized system
- Prevent financial losses by controlling costs and prices
- Automate basic sales-related decisions
- Simulate a business period by managing prices, customers, and suppliers

## Scope
This project focuses mainly on backend development and business logic.

## Technologies (Planned)
- .NET 8
- PostgreSQL
- Angular (optional, future phase)

## Project Status
In initial documentation and planning phase.
